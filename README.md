# Kalman ML

A quantitative trading research framework built in Python.

## Goals

- Strategy-agnostic architecture
- Reproducible research
- Professional software engineering practices
- Unit-tested from the beginning
- Easy transition from research to production

## Planned modules

- Data
- Indicators
- Strategies
- Core backtesting engine
- Machine learning (future)
- Live trading (future)

## installation

```bash
python -m venv .venv
```

Activate the environment.

Install the project:

```bash
pip install -e .[dev]
```

Run tests:

```bash
pytest
```
